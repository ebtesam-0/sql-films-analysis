# SQL Films Analysis 🎬

## 📊 Overview
This project analyzes a films database using SQL to extract insights related to rentals, customers, revenue, staff performance, and film popularity.

## 🛠️ Tools
- SQL
- SQLite / DB Browser

## 🔍 SQL Concepts Used
- JOINs
- GROUP BY and Aggregations
- Subqueries
- Common Table Expressions (CTEs)
- Window Functions

## 📈 Analysis Highlights
- Listed all rented films with rental and return dates
- Calculated total revenue by store
- Identified the top 10 most rented films
- Measured average rental duration per film
- Found customers with no rentals
- Identified the actor appearing in the most films
- Ranked films by revenue within each category
- Calculated monthly revenue growth
- Found films that were never rented
